mikan.aosn.ws
=============

[mikan](https://github.com/mikan) の技術ブログの中身です。[Hugo](https://gohugo.io/) と GitHub Pages を利用しています。

### 基本操作

新しい記事を作る:

```
hugo new post/hello-world.md
vi content/post/hello-world.md
```

ドラフトの表示をチェックする:

```
hugo server -D
```

ドラフトを `false` にして公開する:

```
./deploy.sh
```
